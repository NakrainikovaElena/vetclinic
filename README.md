# Ветклиника

---

Разрабатываемая система поддерживает аутентификацию пользователей на основе логина и пароля, а также авторизацию с использованием ролевой модели доступа. В зависимости от назначенной роли пользователю предоставляется доступ к соответствующему функционалу системы.

---

## Админ
- добавление доктора в систему с выдачей логина и пароля, указанием полного имени, телефона и его специализации.

## Клиент
- добавление питомца с указанием информации о нем: имени, вида и доктора для записи на прием;
- просмотр своих питомцев: имени, вида, лечащего доктора, его специализации и номера телефона;
- добавление нового хозяина, который должен быть зарегистрирован в системе.

## Доктор
- просмотр своих пациентов: имени, вида, их хозяев и диагноза;
- осуществление приема пациента: выбор пациента, записанного на прием и назначение диагноза.

---

Разработка проекта осуществлялась с использованием фреймворка Spring. Для построения серверной части приложения применялись Spring MVC и Spring REST, обеспечивающие реализацию RESTful веб-сервисов. Для взаимодействия с базой данных использовался ORM-фреймворк Hibernate, обеспечивающий объектно-реляционное отображение данных.
